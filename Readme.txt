Test Code assingment:

Precondtions:
Preferrebly Maven 3.6 & above
Java 1.8 and above
git 2.21

To start the test, please navigate to project directory and hit 'mvn clean test'

To start the spring boot application, please navigate to project directory  and hit 'mvn spring-boot:run'

The same can be acheived through IDE by running the cucumber runner class ("EmployeeApiTests.java") & Spring boot application ("Employee Application")

This project is created with following tools:
Java
Maven
Wiremock
Cucumber
RestAssured
Lombok
